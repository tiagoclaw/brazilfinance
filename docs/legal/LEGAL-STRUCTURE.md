## 🛡️ COMPLIANCE E PROTEÇÕES

### LGPD Compliance

**Data Protection Officer (DPO):**
- Obrigatório quando >50K users
- Pode ser terceirizado inicialmente
- Custo: R$ 3-8K/mês

**Privacy by Design:**
- Minimização de dados coletados
- Pseudonimização quando possível  
- Encryption at rest e in transit
- Regular security audits

**Privacy Policy Template:**
```
POLÍTICA DE PRIVACIDADE - BRAZILFINANCE

1. DADOS COLETADOS
- Email, nome, empresa (cadastro)
- Uso da API (logs anonimizados)
- Dados de pagamento (processados via Stripe)

2. FINALIDADE
- Prestação do serviço contratado
- Melhoria do produto
- Comunicação comercial (opt-in)

3. RETENÇÃO
- Dados cadastrais: enquanto conta ativa
- Logs de uso: 24 meses máximo
- Dados pagamento: 5 anos (obrigação fiscal)

4. DIREITOS DO TITULAR
- Acesso, retificação, portabilidade
- Eliminação (exceto obrigações legais)
- Revogação consentimento

5. SEGURANÇA
- Criptografia, controle de acesso
- Auditorias regulares
- Resposta a incidentes

6. CONTATO
dpo@brazilfinance.com.br
```

### Terms of Service

**Principais Cláusulas:**
- **Service Description:** API access, Google Sheets add-on
- **Acceptable Use:** No abuse, scraping, reverse engineering
- **Payment Terms:** Monthly/annual, auto-renewal
- **Service Level:** 99.9% uptime target, maintenance windows
- **Liability:** Limited to amount paid, no consequential damages
- **Termination:** Either party 30 days notice
- **Governing Law:** Lei brasileira, foro São Paulo
