### FAQ - Investor Questions

**Q: Como vocês vão competir com Bloomberg?**
A: Não competimos com Bloomberg diretamente. Eles atendem grandes bancos com orçamento de milhões. Nós democratizamos para PMEs que não podem pagar R$ 10K/mês. Mercados diferentes, propostas diferentes.

**Q: E se o BCB mudar as APIs?**
A: Diversificação de fontes. Temos relacionamentos com 4+ data sources. BCB tem interesse público em disponibilizar dados. Risco baixo, mas mitigado.

**Q: Qual é o moat de longo prazo?**
A: Network effects: quanto mais usuários, mais valiosos nossos insights. Data moat: relacionamentos exclusivos com sources. Integration moat: switching cost alto depois de integrado no workflow.

**Q: Como escalam internacionalmente?**
A: LATAM primeiro (Mexico, Colombia). Mesma base tecnológica, fontes de dados locais. Depois Europa (Espanha, Itália). Modelo replicável.

**Q: Valuation justification?**
A: 2.5x ARR (R$ 1.2M run-rate). Comparables: Quandl vendeu por US$ 200M (15x revenue), YCharts vale US$ 100M+ (10x revenue). Estamos conservadores.
